Linux runnable is added as Freath.
This file requires you to install every png file and keep it under the same location/folder with the executable Freath.
The game requires an executable permission, which can be done through the UI by right clicking on properties, permission or
> chmod +x Freath

# GAMEPLAY
WASD or arrow keys to move and space to jump.
Collect the blueberry and try to avoid the ghost and the spike.
To exit the game, you can simply tap ESC.
Game closes if you die.

# MISSING FEATURES
Ghost chases you around, but doesn't affect the gameplay in any way.
Scaling on different resolutions may be bugged and the blueberry may spawn on unreachable locations due to this bug.
There is no score system.
Spike(s) and the difficulty don't scale over time.
No UI or Menu or save options.
